Paragrafos e quebras de linha

Tudo que estiver entre as tags de p e /p será um paragrafo. Enão se esqueça de que mesmo dando enter as linhas não vão quebrar justamente porque html é uma lingaugem de marcação, ou seja, precisamos adicionar marcas para que ele entenda que precisa quebrar as linhas

Para quebrar o texto em qualquer lugar que precisar
é muito simples, basta usar a tag <br>
que estara udo resolvido

Caso eu colocar algo entre <> será lido como uma tag (caso haja alguma que seja assim)
mas caso mesmo assim que querira usar par alguma demonstração basta usar por exemplo:
<p>



Gambiarra pra pular varias linhas (para pular direito use o css)
